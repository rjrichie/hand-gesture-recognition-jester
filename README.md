# hand-gesture-recognition-jester

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/rjrichie/hand-gesture-recognition-jester.git
```

### 2. Install Dependencies

Create a virtual environment:

```bash
uv venv
source venv/bin/activate    # macOS/Linux
venv\Scripts\activate       # Windows
```

### 3. Run UV

Run the application:

```bash
uv run <filename>           # Will automatically install packages
```

### 4. Add New 

```bash
uv add <package_name>
```
